# 공개 상태

One-Minute Three Kingdom은 현재 제한 초대 베타입니다.

게임은 터미널-first, SSH-first 비동기 MUD로 플레이할 수 있지만, 아직 작은 커뮤니티 테스트를 통해 핵심 루프를 다듬는 중입니다. 이 public repo는 landing page이자 공개 상태판이며, 전체 source release나 uptime 계약이 아닙니다.

## 현재 단계

**제한 초대 베타**

- 접속은 셀프서비스가 아닙니다.
- 플레이어 수용 규모는 의도적으로 작게 둡니다.
- 시즌과 테스트 구간은 초기화될 수 있습니다.
- 주 인터페이스는 SSH 위의 TUI입니다.
- 공개 접속 예시는 placeholder만 사용합니다.

```sh
ssh <player>@<beta-host>
```

## 지금 검증하는 것

- 1분 단위의 짧은 방문이 충분한 진행감으로 이어지는가.
- 보고, 소문, 청원이 세계 상태를 읽기 쉽게 만드는가.
- 비동기 경쟁이 과열되지 않고 공동체의 기억으로 남는가.
- 새 플레이어가 자리를 비운 동안의 변화를 이해할 수 있는가.
- 운영자가 작은 테스트를 지속 가능한 부담으로 운영할 수 있는가.

## 알려진 한계

- 온보딩 문구는 아직 불완전하며 자주 바뀔 수 있습니다.
- 일부 명령어 이름과 도움말은 테스트 사이에 변경될 수 있습니다.
- 밸런스, 진행 속도, 보상은 최종 상태가 아닙니다.
- 베타 시즌 중 플레이어 상태, 저장 데이터, 공개 기록이 초기화될 수 있습니다.
- 현재 단계에서 공개 웹 클라이언트를 약속하지 않습니다.

## 공개 경계

안전과 명확성을 위해 이 public status 문서는 다음을 공개하지 않습니다.

- 실제 베타 host 또는 네트워크 세부 정보
- 운영자 계정명
- 비공개 배포 노트
- 내부 작업 경로
- 미공개 스토리, moderation, 악용 방지 세부 사항

이 정보들은 public showcase repo에 들어오면 안 됩니다.

## 다음 공개 milestone

- 베타 신청 안내를 더 명확히 정리합니다.
- 공개 asset이 준비되면 README의 SVG 이미지 참조를 완성합니다.
- 비공개 운영 단서를 노출하지 않는 선에서 주요 베타 학습을 요약합니다.
- README가 실제 플레이 경험과 계속 일치하도록 유지합니다.
