# 베타 참여

One-Minute Three Kingdom은 현재 초대제로 운영됩니다.

베타를 작게 여는 이유는 단순한 희소성 때문이 아닙니다. 터미널-first 비동기 MUD는 읽기 쉬운 공동 기록, 차분한 온보딩, 플레이어와 운영자 사이의 신뢰가 있어야 오래 갑니다. 천천히 문을 여는 편이 좋은 방이 될 가능성이 높습니다.

## 신청 전 확인할 것

현재 게임의 형태를 먼저 확인해 주세요.

- SSH 터미널/TUI로 플레이합니다.
- 비동기 게임이라 모두가 동시에 접속해 있지 않습니다.
- 베타 시즌 중 초기화될 수 있습니다.
- 일부 시스템은 아직 실험 중입니다.
- 공개 명령 예시는 placeholder입니다.

```sh
ssh <player>@<beta-host>
```

## 신청 방법

공개 신청이 열리면 이 문서의 신청 경로가 실제 링크로 교체됩니다.

그때 요청에 포함하면 좋은 내용:

- 사용하고 싶은 플레이어 이름
- 시간대 또는 주로 접속할 수 있는 시간
- SSH 사용 경험 여부
- 관심 있는 플레이 방식: 정치, 보급, 경쟁, 글쓰기, 탐험, 거친 부분 테스트
- 터미널 플레이에 영향을 주는 접근성 요구

비밀번호, private key, 실제 네트워크 세부 정보, 민감한 개인정보는 보내지 마세요.

## 초대가 오면

현재 테스트 구간에 자리가 있으면 다음 흐름으로 진행됩니다.

1. 운영자가 현재 베타 slice와 맞는지 확인합니다.
2. 승인된 연락 경로로 비공개 초대 정보를 받습니다.
3. 배정된 플레이어 이름과 비공개 host로 접속합니다.
4. 혼란, 버그, 사회적 마찰은 초대 안내에 적힌 공개 또는 비공개 채널로 제보합니다.

접속은 보장되지 않습니다. 조용한 "아직은 아닙니다"는 보통 현재 테스트에 더 작은 인원이 필요하다는 뜻입니다.

## 베타 etiquette

- 비공개 접속 정보는 비공개로 유지합니다.
- 초대 전용 hostname, 계정, moderation note, 미공개 콘텐츠가 보이는 스크린샷을 공개하지 않습니다.
- 게임 안의 경쟁은 가능한 한 fiction으로 다룹니다.
- 악용 가능성, 계정 문제, 괴롭힘은 비공개로 신고합니다.
- 피드백은 실제 플레이 경험 중심으로 남깁니다: 무엇을 시도했고, 무슨 일이 일어났고, 무엇이 헷갈리거나 좋았는지.

## 왜 초대제인가요

이 게임은 reach보다 community를 먼저 만들려고 합니다. 느린 베타는 짧은 터미널 접속만으로도 기억에 남는 공적 역사, 반복해서 마주치는 인물, 부담 없는 협력과 경쟁이 생기는지 확인하기 위한 장치입니다.
