# One-Minute Three Kingdom

![터미널 위에 열린 1분 삼국지 전장](assets/hero-terminal.svg)

**One-Minute Three Kingdom**은 터미널에서 접속하는 SSH-first 비동기 삼국지 MUD입니다.

이 프로젝트는 공개 웹게임이 아닙니다. 지금의 베타는 작고, 텍스트가 많고, 커뮤니티의 리듬을 먼저 검증하는 단계입니다. 플레이어는 짧게 접속해 지난 변화를 읽고, 몇 가지 명령을 내리고, 다른 플레이어가 나중에 이어받을 흔적을 남깁니다.

```sh
ssh <player>@<beta-host>
```

실제 베타 접속 정보는 초대된 테스터에게만 비공개로 전달됩니다.

## 어떤 게임인가요

One-Minute Three Kingdom은 작은 행동이 공적인 기록으로 쌓이는 느린 전략 롤플레잉 공간입니다.

- **TUI MUD**: 주 클라이언트는 브라우저가 아니라 SSH 터미널 세션입니다.
- **비동기 플레이**: 오래 상주하지 않아도 짧은 방문으로 참여할 수 있습니다.
- **커뮤니티 우선**: 목표는 과금 동선이 아니라 서로의 행동을 기억하는 작은 마당입니다.
- **보고, 청원, 소문**: 단순 수치 성장보다 읽히는 정치와 생활감을 중시합니다.
- **시즌 실험**: 베타 시즌 중에는 초기화, 명칭 변경, 시스템 폐기가 일어날 수 있습니다.

## 현재 베타 수준

현재는 제한 초대 베타입니다. 핵심 경험은 테스트할 수 있지만, 누구나 바로 들어오는 공개 서비스로 열기에는 아직 이릅니다.

기대해도 되는 것:

- 명령어와 도움말의 거친 부분
- 아직 완성되지 않은 온보딩
- 짧은 공지 뒤 적용되는 밸런스 변경
- 대규모 이벤트보다 작은 커뮤니티 테스트
- 디자인 실험 종료 후 일어날 수 있는 시즌 초기화

기대하면 안 되는 것:

- 완성된 웹 UI
- 영구 캐릭터 보존 보장
- 공개적인 업타임 보장
- 유료 특전, 꾸미기, 프리미엄 접속권
- 신청 즉시 접속 가능한 셀프서비스 가입

자세한 공개 상태는 [docs/status.md](docs/status.md)를 보세요.

## 베타에 참여하려면

베타 접속은 초대제로 운영됩니다. 커뮤니티가 너무 빨리 커지면 TUI 게임의 장점인 읽기 쉬운 기록, 느린 경쟁, 서로를 알아보는 감각이 먼저 무너질 수 있기 때문입니다.

참여 요청 흐름:

1. [docs/join-beta.md](docs/join-beta.md)를 읽습니다.
2. 공개 신청이 열리면 안내된 경로로 짧은 요청을 보냅니다.
3. 현재 테스트에 맞는 경우, 비공개 초대 정보와 플레이어 이름을 받습니다.

공개 문서의 접속 명령은 항상 placeholder만 사용합니다.

```sh
ssh <player>@<beta-host>
```

비공개 초대 정보는 공개 이슈, 게시글, 스크린샷에 올리지 말아 주세요.

## source repo가 비공개인 이유

이 public repo는 게임을 소개하고, 베타 수준을 설명하고, 참여 신청 경로를 안내하기 위한 showcase입니다. 실제 구현 source repo는 아직 비공개로 둡니다.

그 이유는 다음과 같습니다.

- 미공개 시나리오, 운영 판단, 조정 중인 콘텐츠가 섞여 있습니다.
- 배포와 운영에 관한 가정이 플레이어용 약속으로 오해될 수 있습니다.
- 악용 방지와 moderation 관련 단서는 공개 체크리스트가 되면 위험합니다.
- 아직 폐기될 수 있는 실험 코드가 장기 로드맵처럼 보일 수 있습니다.

영원히 닫겠다는 선언은 아닙니다. 지금 공개해야 할 표면은 게임의 약속, 베타 상태, 참여 방법이며, 전체 제작 작업장은 아직 공개용이 아닙니다.

## 커뮤니티 목표

목표는 많은 사람을 한 번에 모으는 것이 아니라, 플레이어가 서로의 선택을 알아보고 다음 행동을 고민할 수 있는 작은 공동체를 만드는 것입니다.

좋은 베타 피드백은 버그 제보만이 아닙니다. 명령어가 헷갈린 순간, 사회적 마찰이 생긴 장면, 보고서가 읽히지 않은 이유, 다른 플레이어의 선택이 신경 쓰였던 순간도 모두 중요합니다.

기본 원칙:

- 새 플레이어가 무슨 일이 있었는지 따라올 수 있게 남깁니다.
- 경쟁과 원한은 가능한 한 게임 안의 fiction으로 둡니다.
- 악용 가능성, 계정 문제, 괴롭힘은 비공개 경로로 신고합니다.
- 운영자에게 접속 정보, 내부 단서, 우선권을 요구하지 않습니다.
