# Security Policy

This is the public showcase repository for One Minute Three Kingdom. It is meant
for public-facing project information, not for live beta operations.

## Public Issue Scope

You may open a GitHub issue in this repository for:

- Accidental exposure of sensitive information in public docs or assets
- Broken, misleading, or suspicious public links
- Public showcase content that could harm player safety or community trust
- Questions about what may safely be shared from this repository

Please keep reports focused on public materials. Do not include private account
details, invite links, access tokens, server addresses, internal logs, or
step-by-step abuse instructions.

## Live Beta And Account Issues

Real beta access, account problems, invite status, player-specific support, and
live-service security concerns must be handled through the invitation/support
channel where the beta access was provided.

Do not post live beta credentials, account identifiers, private screenshots,
invite-only URLs, exploit details, or service probing results in public issues.
Public issues that contain sensitive details may be removed or redacted.

## Safe Disclosure

If you are unsure whether something belongs in public, treat it as private first.
Do not scan, stress test, scrape, or attempt unauthorized access against any live
service. Report only what is necessary to identify the public-repository problem.
